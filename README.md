## 실행 방법

### 개발 시 사용한 node -v

- **Node.js**: v22.2.0

### 설치 및 실행

1. **의존성 설치**

```bash
npm install
# or
yarn install
# or
pnpm install
```

2. **개발 서버 실행**

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

3. **브라우저에서 확인**
   로컬 실행 후 [http://localhost:3000](http://localhost:3000)에서 확인.
